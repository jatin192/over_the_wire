üìò Dashed Filename

A dashed filename is a normal file whose name starts with a hyphen (-).

Examples:
-
-file
--file

Linux interprets anything starting with '-' as an option/flag,
not as a filename.

________________________________________________________________________________________________________________________________________________________

üìå Special Meanings

-  ‚Üí STDIN / STDOUT  
     Input from keyboard ‚Üí Output to screen  
     (Not a file)

.  ‚Üí Current directory

./- ‚Üí Explicit file path  
      Linux stops treating it as an option


________________________________________________________________________________________________________________________________________________________

‚úÖ Solution

Specify the full or explicit location of the file:

********************************
*  $ cat \"$(pwd)/-\"           *
********************************

üî• 2Ô∏è‚É£ File named `--no-preserve-root`

- Deletes all user data and system files
- System becomes unusable
